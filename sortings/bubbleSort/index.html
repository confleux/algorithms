Bubble sort

Complexity: O(n2)

How does it work: each iteration of the outer for loop smaller or larger elements "bubble" to the top of the list 

 
    void BubbleSort (int * a, int n) {
      for (int i = 0; i < n - 1; ++i) {
        for (int j = 0; j < n - i - 1; ++j) {
          if (a[j] > a[j + 1]) {
            swap(a[j], a[j + 1]);
          }
        }
      }
    }
