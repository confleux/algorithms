Quick sort

Complexity: average O(nlog(n))

How does it work: first we find a pivot, such that it stands on its final psoition, elements from one side are lower than it, element on the another side are larger than it, recursively call QuickSort for elements lower than pivot and for elements bigger than pivot. i in partition is just a swap reference

 
    int Partition (int * a, int left, int right) {
      int pivot = a[right];
      int i = left;
      
      for (int j = left; j < right; ++j) {
        if (a[j] < pivot) {
          swap(a[i], a[j]);
          ++i;
        }
      }
    
      swap(a[i], a[right]);
      return i;
    }
    
    void QuickSort (int * a, int left, int right) {
      if (left < right) {
        int pivot = Partition(a, left, right); 
        QuickSort(a, left, pivot - 1);
        QuickSort(a, pivot + 1, right);
      }
    }
